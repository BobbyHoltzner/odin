# Odin
## A Multiplayer Game Built From Scratch

### To build/run the server
* Navigate to repo in command prompt
* Run shell.bat
* Run builds.bat
* Start with runs.bat
* Debug with debugs.bat

### To build/run the client
* Navigate to repo in command prompt
* Run shell.bat
* Run buildc.bat
* Start with runc.bat
* Debug with debugc.bat
